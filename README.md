# COMP-3600 Computer Graphics
This repository is for saving my work for that class. Most of them are WinForms (which is depracated) projects

## Why C# ?
Visual Basic is essentially depracated and it's recommended to use more modern languages.
For this case C# has the same Graphics library, so it has the exact same methods you would find in VB like 
```cs 
e.Graphics.DrawEllipse()
e.Graphics.DrawRectangle()
e.Graphics.FillPolygon()
```
etc.

C# also helps me structure bigger programs better like [the owl project](https://github.com/arosario513/COMP-3600/tree/main/owl) which is drawn in parts.
